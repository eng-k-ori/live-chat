定義をしたコンテナを起動
================================================================================================
docker-composeとDockerFileの、CMD ["npm", "run", "serve"]をコメントアウト
================================================================================================
docker-compose up
================================================================================================
docker exec -it <コンテナ名> sh
vueのコンテナ内に入る
================================================================================================
vue create .
------------------------------------------------------------------------------------------------
現在いるディレクトリにプロジェクトを作成するかどうかの確認メッセージが表示されます。Yと入力し、Enterキーを押下しま
す。（YはYesのYです。）
次はどのVueバージョンで作成するかが聞かれます。下矢印キーを押してDefault (Vue 3) ([Vue 3] babel, eslint)を
選択し、Enterキーを押下します。
次にパッケージマネージャの選択肢を聞かれます。Use NPMを選択し、Enterキーを押下します。
================================================================================================
Vue.jsのインストールが完了したら、今度はVue Routerを導入していきます。
$ vue add router
------------------------------------------------------------------------------------------------
下記のようにhistory modeで使用するかどうかの確認メッセージが表示されるので、Yと入力しEnterを押してください
================================================================================================
Vue Routerのインストールが完了したら、下記コマンドを実行してください。
$ npm run serve
================================================================================================
docker-composeとDockerFileの、CMD ["npm", "run", "serve"]を元に戻す

################################################################################################
Vue.jsでAction Cableを使用するためのライブラリをインストールしていきます。
$ npm install --save actioncable
$ npm run serve

################################################################################################
Font Awesomeを使用できる様に、専用のライブラリをインストール
$ npm i --save @fortawesome/vue-fontawesome@prerelease
$ npm i --save @fortawesome/fontawesome-svg-core
$ npm i --save @fortawesome/free-solid-svg-icons
